## 🧰 ncexs Toolkit – v3.1 Обновление Мастерства Системы

### ✨ Особенности и Улучшения
- **Улучшенная информация о системе**: Добавлены сведения о материнской плате, сборке ОС, времени работы и серийном номере.
- **Мощная очистка**: Теперь очищает кэш Windows Update, CrashDumps, WER и логи CBS.
- **Очистка кэша приложений**: Безопасная очистка кэша Discord, Spotify, Steam, Telegram и WhatsApp.
- **Очистка журналов событий**: Новое меню (16) для очистки логов событий Windows.
- **Живые метрики**: Теперь инструменты показывают, сколько МБ было освобождено.
- **Безопасная проверка**: Добавлена проверка `chkdsk C: /scan` наряду с SFC и DISM.

### 🎨 UI и UX
- **Сетка из 17 меню**: Интерфейс идеально адаптирован под нечетное количество пунктов (17).
- **Язык по умолчанию**: Английский (EN).
- **Удалено**: Опция сброса Windows Update убрана в целях безопасности системы.

# 🚀 ncexs Toolkit v3.0.1

## 🧰 ncexs Toolkit – v3.0.1 Улучшения безопасности и UX

### ✨ Новые возможности и улучшения
- **Безопасность интерфейса**: Добавлены четкие динамические мультиязычные инструкции по отмене операций, помогающие пользователям безопасно прерывать выполнение задач без резкого закрытия окна консоли.
- **Корректный перехват Ctrl+C**: Реализован глобальный обработчик исключений (`trap [PipelineStoppedException]`) в главном цикле. Нажатие `Ctrl+C` для отмены текущей задачи больше не приводит к выходу из утилиты в оболочку. Скрипт корректно перехватывает сигнал отмены, выводит локализованное сообщение и возвращает пользователя в главное меню без необходимости перезапуска скрипта.
- **Мультиязычная поддержка**: Добавлены строки перевода `"Process_Cancel"` и `"Process_Aborted"` для английской ("EN") и индонезийской ("ID") локализаций.
- **Контекстные предупреждения**: Предупреждение об отмене интегрировано в стандартное окно подтверждения перед запуском очистки, дефрагментации, сжатия ОС, обновления ПО и бэкапа драйверов.

# 🚀 ncexs Toolkit v3.0

## 🎓 ncexs Toolkit – v3.0 Сборка после промежуточных экзаменов

### ⚡ Запуск без скачивания файлов
- **Никаких загрузок**: Начиная с версии v3.0, больше не нужно скачивать files `.ps1` вручную. Скрипт запускается напрямую из ветки `main` одной командой PowerShell.

### ✨ Новые возможности и улучшения
- **Дизайн и интерфейс**: ASCII-арт главного меню заменен на объемный 3D-дизайн 'Big Money' синего цвета. Пункт смены языка выделен желтым цветом.
- **Локализация**: Полный перевод интерфейса на английский и индонезийский языки с использованием функции `Get-Translation`.
- **Быстрое переключение языка**: Переключение языка клавишей [L]. Исправлен баг двойного срабатывания переключателя.
- **Переключатель DNS**: Интерактивное подменю с 5 популярными DNS-провайдерами.
- **Детальная статистика**: Отображение освобожденных мегабайт "До и после" для ОЗУ и очистки диска.
- **Восстановление сети**: Пошаговые уведомления о прогрессе сброса сетевых настроек.

### 🐛 Исправления ошибок и рефакторинг
- **Циклы меню**: Устранено зацикливание меню добавлением очистки экрана и стандартных обработчиков.
- **Продвинутый деинсталлятор**: Добавлена пауза после вывода результатов деинсталляции.
- **Абстракция строк**: Все текстовые строки вынесены в глобальный словарь переводов.
