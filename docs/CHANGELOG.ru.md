## 🧰 ncexs Toolkit – Обновление v3.5.1 Hotfix

### 🐛 Исправления ошибок и рефакторинг
- **Исправление оптимизатора ОЗУ**: Устранена синтаксическая ошибка динамической компиляции C# (блок catch в стиле PowerShell и `Write-Debug` внутри строки C#), приводившая к сбою Меню 5 (Оптимизатор памяти) при запуске.
- **Резервный перевод**: Улучшена функция `Get-Translation` для автоматического переключения на английский язык (EN), если ключ отсутствует в словаре выбранного языка.
- **Исправление заголовка DNS Changer**: Исправлено несоответствие заголовка в меню DNS Changer, где ранее отображалось «Очистка кэша приложений» вместо «Настройка DNS».
- **Поддержка двоеточия в SSID / паролях Wi-Fi**: Исправлено усечение имен или паролей Wi-Fi, содержащих символ двоеточия (`:`), в утилите восстановления Wi-Fi.
- **Точная статистика очистки**: Логика удаления файлов изменена так, чтобы учитывать размер только успешно удаленных файлов, предотвращая неверную статистику из-за заблокированных файлов.
- **Защита индекса деинсталлятора**: Добавлена проверка границ индекса в интерактивном деинсталляторе для предотвращения ошибок выхода за пределы массива.

## 🎓 ncexs Toolkit – v3.5 UAS Freedom Update

### ✨ Функции и улучшения
- **Глубокая очистка системы и приложений**: Объединение системного мусора, кеша приложений (поддержка Teams, Zoom, Skype, Epic Games, GOG, WhatsApp, Discord, Spotify, Steam, Telegram) и журналов событий Windows в одну утилиту глубокой очистки (`[11]`). Поддержка автоматического (тихого) запуска в рамках «Обслуживания в один клик».
- **Диагностика системы и меню отчетов**: Создано специальное подменю диагностики (`[19]`), содержащее экспорт отчета аудита ПК в HTML, отчет о состоянии батареи, проверку активации лицензии и проверку совместимости с Windows 11.

- **Интеграция диспетчера автозагрузки**: Добавлен интерактивный диспетчер (`[18]`) для просмотра, включения или отключения приложений автозапуска в реестре и папках автозагрузки.
- **Логически классифицированный сетчатый интерфейс**: Переструктурирована 2-колоночная разметка меню на упорядоченные разделы: левая колонка для диагностики и системных утилит, правая колонка для производительности, очистки и оптимизации.
- **Конвейеры CI/CD и безопасности**: Интегрирован PSScriptAnalyzer для статического анализа кода, Pester для изолированного юнит-тестирования и VirusTotal CLI Action для автоматического сканирования на наличие вредоносных программ при каждом коммите.

## 🧰 ncexs Toolkit – v3.3 Обновление производительности и диагностики

### ✨ Функции и улучшения
- **Оценки производительности WinSAT**: Интегрированы метрики средства оценки системы Windows (WinSAT) в экран CLI «Сведения о системе» и интерактивный HTML-отчет об аудите ПК. Имеет визуальные цветные шкалы рейтинга (`█` и `░`), указывающие на производительность подсистем (процессор, оперативная память, видеокарта, игровая графика, диск).
- **Состояние батареи в сведениях о системе**: Встроены надежные показатели состояния батареи в режиме реального времени (процент износа, расчетная емкость по сравнению с емкостью при полной зарядке) непосредственно в экран CLI «Сведения о системе» (автоопределение на ноутбуках).
- **Масштабная оптимизация производительности**:
  - **Суперзаряженный очиститель кэша браузера**: Оптимизировано разрешение путей для выполнения на уровне папок вместо сканирования отдельных файлов. Очищает кэш глубокого уровня до 100 раз быстрее.
  - **Обход компиляции при очистке ОЗУ**: Реализованы активные проверки типов для пропуска компиляции C# при последующих очистках ОЗУ, что устраняет задержку выполнения.

### 🐛 Исправления ошибок и рефакторинг
- **Надежный парсер отчетов о батарее**: Обновлен шаблон регулярных выражений (`DESIGN CAPACITY.*?>\s*([\d,.]+)\s*mWh`) для безопасной поддержки современных HTML-отчетов Windows 11 и динамической обработки разделителей тысяч.
- **Совместимость с PowerShell 3.0**: Исправлены вызовы ускорителей типов .NET (`[System.Net.ServicePointManager]`) в командах быстрого запуска во всех документах README для поддержки устаревших сред.

# 🚀 ncexs Toolkit v3.2

## 🧰 ncexs Toolkit – v3.2 Обновление обслуживания и аудита системы

### ✨ Особенности и Улучшения
- **Центр обслуживания ПК**: Добавлено новое комплексное подменю (Меню 17):
  - **Отчет об аудите ПК (HTML)**: Создает интерактивный HTML-отчет о характеристиках ПК на Рабочем столе. Анализирует CPU, RAM, ОС, материнскую плату и **активные учетные записи Windows** с указанием привилегий (Администратор/Стандартный).
  - **Обслуживание в один клик**: Выполняет авто-обслуживание (очистка мусора, сброс DNS, SFC/DISM), бережно сохраняя корзину (Recycle Bin).
  - **Анализ здоровья батареи**: Измеряет износ батареи в реальном времени или определяет настольный ПК.
  - **Управление удаленным рабочим столом (RDP)**: Включает/выключает RDP и настраивает брандмауэр.
  - **Очистка кэша браузеров**: Очищает кэш Google Chrome, Microsoft Edge и Mozilla Firefox.
- **Мгновенная очистка логов через .NET**: Очистка Event Viewer ускорена до < 1 секунды благодаря прямому использованию API .NET (в 60 раз быстрее).
- **DNS на всех адаптерах**: Настройки DNS теперь применяются ко всем активным физическим сетевым адаптерам одновременно.
- **Двухсторонний Compact OS**: Добавлена поддержка как сжатия (`/CompactOS:always`) и разжатия (`/CompactOS:never`) системных файлов.

### 🐛 Исправление ошибок и локализация
- **Локализация и Безопасность**: Исправлен баг смешения языков в Compact OS благодаря полной интеграции с `$global:Translations`.
- **Защита пользовательских файлов**: Процедура авто-очистки теперь не затрагивает Корзину, предотвращая случайную потерю важных файлов.

# 🚀 ncexs Toolkit v3.0.1

## 🧰 ncexs Toolkit – v3.0.1 Улучшения безопасности и UX

### ✨ Новые возможности и улучшения
- **Безопасность интерфейса**: Добавлены четкие динамические мультиязычные инструкции по отмене операций, помогающие пользователям безопасно прерывать выполнение задач без резкого закрытия окна консоли.
- **Корректный перехват Ctrl+C**: Реализован глобальный обработчик исключений (`trap [PipelineStoppedException]`) в главном цикле. Нажатие `Ctrl+C` для отмены текущей задачи больше не приводит к выходу из утилиты в оболочку. Скрипт корректно перехватывает сигнал отмены, выводит локализованное сообщение и возвращает пользователя в главное меню без необходимости перезапуска скрипта.
- **Мультиязычная поддержка**: Добавлены строки перевода `"Process_Cancel"` и `"Process_Aborted"` для английской ("EN") и индонезийской ("ID") локализаций.
- **Контекстные предупреждения**: Предупреждение об отмене интегрировано в стандартное окно подтверждения перед запуском очистки, дефрагментации, сжатия ОС, обновления ПО и бэкапа драйверов.

# 🚀 ncexs Toolkit v3.0

## 🎓 ncexs Toolkit – v3.0 Сборка после промежуточных экзаменов

### ⚡ Запуск без скачивания файлов
- **Никаких загрузок**: Начиная с версии v3.0, больше не нужно скачивать files `.ps1` вручную. Скрипт запускается напрямую из ветки `main` одной командой PowerShell.

### ✨ Новые возможности и улучшения
- **Дизайн и интерфейс**: ASCII-арт главного меню заменен на объемный 3D-дизайн 'Big Money' синего цвета. Пункт смены языка выделен желтым цветом.
- **Локализация**: Полный перевод интерфейса на английский и индонезийский языки с использованием функции `Get-Translation`.
- **Быстрое переключение языка**: Переключение языка клавишей [L]. Исправлен баг двойного срабатывания переключателя.
- **Переключатель DNS**: Интерактивное подменю с 5 популярными DNS-провайдерами.
- **Детальная статистика**: Отображение освобожденных мегабайт "До и после" для ОЗУ и очистки диска.
- **Восстановление сети**: Пошаговые уведомления о прогрессе сброса сетевых настроек.

### 🐛 Исправления ошибок и рефакторинг
- **Циклы меню**: Устранено зацикливание меню добавлением очистки экрана и стандартных обработчиков.
- **Продвинутый деинсталлятор**: Добавлена пауза после вывода результатов деинсталляции.
- **Абстракция строк**: Все текстовые строки вынесены в глобальный словарь переводов.
