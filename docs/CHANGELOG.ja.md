## 🧰 ncexs Toolkit – v3.0.1 安全性＆UXの改善

### ✨ 新機能と強化
- **UX 安全性の改善**: 非技術系のユーザーがターミナルウィンドウを強制終了することなく安全にタスクをキャンセルできるよう、明確で動的な多言語キャンセル手順を追加しました。
- **Ctrl+C の適切な捕捉**: メインループ内にグローバルな例外ハンドラー (`trap [PipelineStoppedException]`) を実装しました。実行中のタスクを中止するために `Ctrl+C` を押しても、ツールキットがシェルに終了することはなくなりました。スクリプトは中止シグナルを適切にキャッチし、キャンセルメッセージを表示して、コマンドを再実行することなくメインメニューに戻ります。
- **多言語サポート**: 英語 ("EN") およびインドネシア語 ("ID") 用の新しい `"Process_Cancel"` および `"Process_Aborted"` 翻訳文字列を実装し、いつでも `Ctrl+C` を使用してプロセスをキャンセルできるようガイドします。
- **コンテキスト警告プロンプト**: ジャンククリーナー、デフラグ、Compact OS などのプロセスを実行する前、および `winget` 更新やドライバーバックアップを開始する直前に自動的に表示される標準の確認プロンプトにキャンセル警告を統合しました。

# 🚀 ncexs Toolkit v3.0

## 🎓 ncexs Toolkit – v3.0 中間テスト後ビルド

### ⚡ 直接実行 (リリースファイル不要)
- **ダウンロード不要**: v3.0 から、リリース画面から `.ps1` ファイルを手動でダウンロードする必要がなくなりました。ツールキットは1つの PowerShell コマンドで `main` ブランチから直接実行されます。

### ✨ 新機能と強化
- **UI＆デザイン**: メインの ASCII アートを太字の 3D 'Big Money' デザインに変更し、青色のテーマを採用しました。「言語変更」オプションの視認性を向上させるため、黄色に変更しました。
- **ローカリゼーション**: 英語およびインドネシア語の完全な翻訳カバレッジを追加しました。ハードコードされた文字列を `Get-Translation` に置き換えました。
- **言語切り替え**: [L] キーによる英語とインドネシア語の素早い切り替えを実装しました。
- **DNS チェンジャー**: 5つの主要プロバイダーを備えた対話型 DNS チェンジャーのサブメニューを追加しました。
- **詳細な最適化指標**: ジャンククリーナーおよびメモリ最適化ツールに「実行前／実行後」の MB 統計を追加しました。
- **ネットワーク修復**: ステップバイステップの進捗通知を追加しました。

### 🐛 バグ修正とリファクタリング
- **メニューループ**: switch 構文に Clear-Host および default ケースを追加し、無限ループを修正しました。
- **高度なアンインストーラー**: 終了時の停止 (pause) を追加しました。
- **文字列の抽象化**: すべてのハードコードされた文字列をグローバル翻訳辞書にまとめました。
