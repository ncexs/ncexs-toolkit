## 🎓 ncexs Toolkit – v3.5 UAS Freedom Update

### ✨ 機能と向上
- **システムとアプリのディープクリーナー**: システムのジャンク、アプリのキャッシュ（Teams、Zoom、Skype、Epic Games、GOG、WhatsApp、Discord、Spotify、Steam、Telegramをサポート）、およびWindowsイベントログを1つのディープクリーニングユーティリティ（`[11]`）に統合。「ワンクリックメンテナンス」内でのサイレント実行をサポート。
- **システム診断とレポートメニュー**: PC監査HTMLエクスポート、バッテリーヘルスレポート、ライセンス認証チェック、およびWindows 11互換性チェックを含む、専用の診断サブメニュー（`[19]`）を作成。

- **論理的に分類されたグリッドUI**: 2列のメニューレイアウトを整理されたセクションに再構成：左列は診断とシステムユーティリティ、右列はパフォーマンス、クリーニング、および最適化。
- **CI/CDとセキュリティパイプライン**: 静的コード分析用のPSScriptAnalyzer、サンドボックス化されたユニットテスト用のPester、およびコードプッシュごとの自動マルウェアスキャンのためのVirusTotal CLI Actionを統合しました。

## 🧰 ncexs Toolkit – v3.3 パフォーマンスと診断のアップデート

### ✨ 機能と機能向上
- **WinSAT パフォーマンス スコア**: Windows システム評価ツール (WinSAT) のメトリクスをシステム情報 CLI 画面とインタラクティブな HTML PC 監査レポートに統合しました。サブシステム (CPU、RAM、GPU、ゲーム用グラフィックス、ディスク) のパフォーマンスを示す視覚的な色付きレーティング バー (`█` および `░`) を備えています。
- **システム情報でのバッテリー正常性**: リアルタイムのバッテリー正常性メトリクス (摩耗率、設計容量とフル充電容量) をシステム情報 CLI 画面に直接埋め込みました (ノート PC では自動検出)。
- **大幅なパフォーマンス最適化**:
  - **強力なブラウザ キャッシュ クリーナー**: 個々のファイルをスキャンする代わりにフォルダー レベルで実行するようにパス解決を最適化しました。ディープレベルのキャッシュを最大 100 倍高速にクリアします。
  - **RAM クリーン コンパイル バイパス**: 以降の RAM クリーンアップで C# のコンパイルをスキップするアクティブな型チェックを実装し、実行時の遅延を排除しました。

### 🐛 バグ修正とリファクタリング
- **堅牢なバッテリー レポート パーサー**: 正規表現パターン (`DESIGN CAPACITY.*?>\s*([\d,.]+)\s*mWh`) をアップグレードし、最新 of Windows 11 HTML レポートを安全にサポートし、桁区切り文字を動的に処理できるようにしました。
- **PowerShell 3.0 の互換性**: レガシー環境をサポートするために、すべての README ドキュメントのクイック起動コマンドにおける .NET 型アクセラレータの呼び出し (`[System.Net.ServicePointManager]`) を修正しました。

# 🚀 ncexs Toolkit v3.2

## 🧰 ncexs Toolkit – v3.2 システム保守・監査アップデート

### ✨ 機能と強化
- **PCメンテナンススイート**: 新しい包括的なサブメニュー（メニュー17）を追加しました：
  - **PC監査レポート (HTML)**: デスクトップに洗練されたインタラクティブなHTML仕様レポートを作成します。CPU、RAM、OS、マザーボード、および**有効なWindowsアカウント**（管理者権限/一般権限バッジ付き）を調査します。
  - **ワンクリックメンテナンス**: ゴミ箱を保護しつつ、自動システム保守（ジャンク消去、DNSフラッシュ、SFC/DISMスキャン）を実行します。
  - **バッテリー寿命アナライザー**: リアルタイムでバッテリー摩耗率を計算するか、デスクトップPCを検出します。
  - **リモートデスデスクトップ (RDP) 管理**: RDPの有効/無効化とファイアウォール規則の構成を行います。
  - **ブラウザキャッシュクリーナー**: Chrome、Edge、Firefoxのキャッシュをクリアします。
- **.NET高速イベントログ消去**: .NET APIを直接使用してイベントログのクエリと消去を行い、1秒未満で高速実行します。
- **マルチアダプターDNS**: 設定されたDNSをすべての有効な物理ネットワークアダプターに同時に適用します。
- **双方向Compact OS**: システムファイルの圧縮（`/CompactOS:always`）と解凍（`/CompactOS:never`）の両方をサポート。

### 🐛 不具合修正と多言語対応
- **ローカライズと安全性**: Compact OSの言語混在バグを完全に修正し、翻訳システムに統合しました。
- **ユーザーファイルの保護**: 自動メンテナンス時の意図しないファイル消失を防ぐため、ゴミ箱のクリア処理を除外しました。

# 🚀 ncexs Toolkit v3.0.1

## 🧰 ncexs Toolkit – v3.0.1 安全性＆UXの改善

### ✨ 新機能と強化
- **UX 安全性の改善**: 非技術系のユーザーがターミナルウィンドウを強制終了することなく安全にタスクをキャンセルできるよう、明確で動的な多言語キャンセル手順を追加しました。
- **Ctrl+C の適切な捕捉**: メインループ内にグローバルな例外ハンドラー (`trap [PipelineStoppedException]`) を実装しました。実行中のタスクを中止するために `Ctrl+C` を押しても、ツールキットがシェルに終了することはなくなりました。スクリプトは中止シグナルを適切にキャッチし、キャンセルメッセージを表示して、コマンドを再実行することなくメインメニューに戻ります。
- **多言語サポート**: 英語 ("EN") およびインドネシア語 ("ID") 用の新しい `"Process_Cancel"` および `"Process_Aborted"` 翻訳文字列を実装し、いつでも `Ctrl+C` を使用してプロセスをキャンセルできるようガイドします。
- **コンテキスト警告プロンプト**: ジャンククリーナー、デフラグ、Compact OS などのプロセスを実行する前、および `winget` 更新やドライバーバックアップを開始する直前に自動的に表示される標準の確認プロンプトにキャンセル警告を統合しました。

# 🚀 ncexs Toolkit v3.0

## 🎓 ncexs Toolkit – v3.0 中間テスト後ビルド

### ⚡ 直接実行 (リリースファイル不要)
- **ダウンロード不要**: v3.0 から、リリース画面から `.ps1` ファイルを手動でダウンロードする必要がなくなりました。ツールキットは1つの PowerShell コマンドで `main` ブランチから直接実行されます。

### ✨ 新機能と強化
- **UI＆デザイン**: メインの ASCII アートを太字の 3D 'Big Money' デザインに変更し、青色のテーマを採用しました。「言語変更」オプションの視認性を向上させるため、黄色に変更しました。
- **ローカリゼーション**: 英語およびインドネシア語の完全な翻訳カバレッジを追加しました。ハードコードされた文字列を `Get-Translation` に置き換えました。
- **言語切り替え**: [L] キーによる英語とインドネシア語の素早い切り替えを実装しました。
- **DNS チェンジャー**: 5つの主要プロバイダーを備えた対話型 DNS チェンジャーのサブメニューを追加しました。
- **詳細な最適化指標**: ジャンククリーナーおよびメモリ最適化ツールに「実行前／実行後」の MB 統計を追加しました。
- **ネットワーク修復**: ステップバイステップの進捗通知を追加しました。

### 🐛 バグ修正とリファクタリング
- **メニューループ**: switch 構文に Clear-Host および default ケースを追加し、無限ループを修正しました。
- **高度なアンインストーラー**: 終了時の停止 (pause) を追加しました。
- **文字列の抽象化**: すべてのハードコードされた文字列をグローバル翻訳辞書にまとめました。
