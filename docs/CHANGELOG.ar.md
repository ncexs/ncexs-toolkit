## 🎓 ncexs Toolkit – v3.5 UAS Freedom Update

### ✨ الميزات والتحسينات
- **منظف النظام والتطبيقات العميق**: دمج ملفات النظام المؤقتة (Junk)، وذاكرة التخزين المؤقت للتطبيقات (يدعم Teams، Zoom، Skype، Epic Games، GOG، WhatsApp، Discord، Spotify، Steam، Telegram)، وسجلات أحداث Windows في أداة تنظيف عميق واحدة (`[11]`). دعم التشغيل الصامت داخل الصيانة بنقرة واحدة.
- **قائمة تشخيصات النظام والتقارير**: إنشاء قائمة فرعية مخصصة للتشخيصات (`[19]`) تحتوي على تصدير تقرير فحص الكمبيوتر بصيغة HTML، وتقرير صحة البطارية، والتحقق من تنشيط الترخيص، وفحص التوافق مع Windows 11.
- **أداة فحص التوافق مع Windows 11**: دمج أداة فحص التوافق للأجهزة الأساسية للتحقق من وضع التمهيد UEFI، وحالة الإقلاع الآمن (Secure Boot)، ومواصفات TPM 2.0 مع أدلة توصيات مفصلة.
- **تكامل مدير برامج بدء التشغيل**: إضافة مدير تفاعلي (`[18]`) لعرض برامج بدء تشغيل سجل النظام (Registry) والمجلدات، وتمكينها أو تعطيلها.
- **واجهة مستخدم شبكية مصنفة منطقيًا**: إعادة هيكلة تخطيط القائمة المكونة من عمودين إلى أقسام منظمة: العمود الأيسر للتشخيصات وأدوات النظام، والعمود الأيمن للأداء والتنظيف والتحسين.
- **مسارات التطوير والتحقق المستمر (CI/CD) والأمان**: تم دمج PSScriptAnalyzer لتحليل التعليمات البرمجية الثابتة، وPester لاختبارات الوحدات المعزولة، وVirusTotal CLI Action لفحص البرامج الضارة تلقائيًا عند كل عملية دفع للتعليمات البرمجية.

# 🚀 ncexs Toolkit v3.3

## 🧰 ncexs Toolkit – v3.3 تحديث الأداء والتشخيص

### ✨ الميزات والتحسينات
- **درجات أداء WinSAT**: دمج مقاييس أداة تقييم نظام التشغيل Windows (WinSAT) في شاشة معلومات النظام CLI وتقرير فحص الكمبيوتر التفاعلي HTML. يتميز بأشرطة تقييم ملونة مرئية (`█` و `░`) تشير إلى أداء الأنظمة الفرعية (المعالج، الذاكرة العشوائية، كرت الشاشة، رسومات الألعاب، القرص الصلب).
- **صحة البطارية في معلومات النظام**: تضمين مقاييس صحة البطارية في الوقت الفعلي (نسبة التآكل، السعة التصميمية مقابل سعة الشحن الكامل) مباشرة في شاشة معلومات النظام CLI (الكشف التلقائي على أجهزة الكمبيوتر المحمولة).
- **تحسينات هائلة في الأداء**:
  - **منظف كاش المتصفح الفائق**: تم تحسين دقة المسار ليتم تنفيذه على مستوى المجلد بدلاً من فحص الملفات الفردية. ينظف ذاكرة التخزين المؤقت العميقة أسرع بـ 100 مرة.
  - **تجاوز تجميع تنظيف الذاكرة**: تم تطبيق فحوصات نوع النشط لتخطي تجميع C# في عمليات تنظيف الذاكرة اللاحقة، مما يزيل تأخر التنفيذ.

### 🐛 إصلاح الأخطاء وإعادة الهيكلة
- **محلل تقرير بطارية قوي**: ترقية نمط التعبير المنتظم (`DESIGN CAPACITY.*?>\s*([\d,.]+)\s*mWh`) لدعم تقارير Windows 11 HTML الحديثة بأمان ومعالجة فواصل الآلاف ديناميكيًا.
- **توافق PowerShell 3.0**: تصحيح استدعاءات مسرع نوع .NET (`[System.Net.ServicePointManager]`) في أوامر التشغيل السريع عبر جميع مستندات README لدعم البيئات القديمة.

# 🚀 ncexs Toolkit v3.2

## 🧰 ncexs Toolkit – v3.2 تحديث صيانة وفحص النظام

### ✨ الميزات والتحسينات
- **حزمة صيانة الكمبيوتر**: تمت إضافة قائمة فرعية شاملة وجديدة (قائمة 17):
  - **تقرير فحص الكمبيوتر (HTML)**: يولد تقريراً تفاعلياً جميلاً لمواصفات الكمبيوتر على سطح المكتب. يفحص المعالج، والرام، ونظام التشغيل، واللوحة الأم، و **حسابات ويندوز النشطة** مع توضيح مستوى الصلاحيات (مسؤول/قياسي).
  - **صيانة بنقرة واحدة**: صيانة تلقائية بالكامل (تنظيف الملفات، مسح DNS، فحص SFC/DISM) مع الحفاظ بأمان على سلة المهملات.
  - **محلل صحة البطارية**: يقيس نسبة تآكل البطارية الفعلي أو يكتشف الكمبيوتر المكتبي تلقائياً.
  - **إدارة سطح المكتب البعيد (RDP)**: تفعيل أو تعطيل RDP وتكوين جدار الحماية فوراً.
  - **منظف ذاكرة التخزين المؤقت للمتصفحات**: تنظيف كاش متصفحات Chrome و Edge و Firefox.
- **تنظيف سجلات الأحداث فائق السرعة عبر .NET**: استخدام مباشر لـ .NET API لمسح جميع السجلات في أقل من ثانية واحدة (أسرع بـ 60 مرة).
- **DNS لجميع المحولات**: تطبيق إعدادات DNS على كافة محولات الشبكة النشطة في نفس الوقت.
- **أداة Compact OS ثنائية الاتجاه**: دعم ضغط ملفات النظام (`/CompactOS:always`) وإلغاء ضغطها (`/CompactOS:never`).

### 🐛 إصلاح الأخطاء والتعريب
- **إصلاح الأخطاء والتعريب**: معالجة مشكلة تداخل اللغات في أداة Compact OS وترجمتها بالكامل في نظام الترجمات.
- **حماية ملفات المستخدم**: استثناء سلة المهملات من الصيانة التلقائية لمنع حذف الملفات المهمة دون قصد.

## 🧰 ncexs Toolkit – v3.1 تحديث إتقان النظام

- **منظف أكثر قوة**: تنظيف ذاكرة التخزين المؤقت لـ Windows Update و CrashDumps و WER وسجلات CBS.

- **فحص أكثر أمانًا**: إضافة `chkdsk C: /scan` جنبًا إلى جنب مع SFC و DISM.

- **شبكة من 17 قائمة**: واجهة محسّنة تدعم 17 عنصرًا (رقم فردي) بدقة.

# 🚀 ncexs Toolkit v3.0.1

## 🧰 ncexs Toolkit – v3.0.1 تحسينات الأمان وتجربة المستخدم

### ✨ الميزات والتحسينات
- **تحسين أمان تجربة المستخدم**: إضافة إرشادات إلغاء واضحة وديناميكية متعددة اللغات لمساعدة المستخدمين غير التقنيين على فهم كيفية إلغاء المهام بأمان دون إغلاق نافذة الطرفية بشكل مفاجئ.
- **التعامل السلس مع Ctrl+C**: تطبيق معالج استثناءات شامل (`trap [PipelineStoppedException]`) داخل الحلقة الرئيسية. الضغط على `Ctrl+C` لإلغاء مهمة قيد التشغيل لن يؤدي بعد الآن إلى الخروج من الأداة إلى الغلاف (Shell). يلتقط النص البرمجي الآن إشارة الإلغاء بسلاسة، ويعرض رسالة إلغاء مخصصة، ويعيد المستخدم إلى القائمة الرئيسية دون الحاجة لإعادة تشغيل أمر التنفيذ.
- **دعم متعدد اللغات**: تطبيق نصوص ترجمة جديدة `"Process_Cancel"` و `"Process_Aborted"` تشمل اللغتين الإنجليزية ("EN") والإندونيسية ("ID")، لتوجيه المستخدمين لإلغاء العمليات باستخدام `Ctrl+C` في أي وقت.
- **رسائل التحذير السياقية**: دمج تحذير الإلغاء في نافذة التأكيد القياسية ليتم عرضه تلقائياً قبل تنفيذ أي عملية (مثل تنظيف الملفات غير الضرورية، وإلغاء التجزئة، و Compact OS)، وقبل بدء ترقيات `winget` والنسخ الاحتياطي للتعريفات.

# 🚀 ncexs Toolkit v3.0

## 🎓 ncexs Toolkit – v3.0 إصدار ما بعد اختبارات منتصف الفصل

### ⚡ التنفيذ المباشر (بدون ملفات تنزيل)
- **لا يلزم التنزيل**: بدءاً من v3.0، تم القضاء على الحاجة لتنزيل ملفات `.ps1` يدوياً من صفحة الإصدارات. تعمل الأداة الآن مباشرة من الفرع `main` عبر أمر PowerShell واحد، مما يضمن تشغيل أحدث كود برمجياً فوراً.

### ✨ الميزات والتحسينات
- **واجهة المستخدم والتصميم**: استبدال الفن النصي ASCII الرئيسي بتصميم ثلاثي الأبعاد 'Big Money' باللون الأزرق. تحسين ظهور خيار "تغيير اللغة" عبر تغيير لونه إلى الأصفر.
- **التعريب**: إضافة تغطية كاملة للترجمة الإنجليزية والإندونيسية. استبدال جميع النصوص الثابتة في وظائف الأداة بـ `Get-Translation`.
- **تبديل اللغة السريع**: تطبيق ميزة التبديل السريع للغة [L] بين الإنجليزية والإندونيسية.
- **مُبدّل DNS**: إضافة قائمة فرعية تفاعلية لتغيير DNS تضم 5 مزودين رئيسيين.
- **مقاييس تحسين تفصيلية**: إضافة إحصائيات الميجابايت "قبل وبعد" إلى منظف الملفات ومُحسّن الذاكرة.
- **إصلاح الشبكة**: إضافة إشعارات تقدم خطوة بخطوة للعمليات.

### 🐛 إصلاح الأخطاء وإعادة الهيكلة
- **حلقات القائمة**: إصلاح حلقات القائمة اللانهائية بإضافة Clear-Host والحالات الافتراضية.
- **أداة إلغاء التثبيت المتقدمة**: إضافة إيقاف مؤقت (pause) مفقود في نتائج أداة إلغاء التثبيت.
- **تجريد النصوص**: نقل النصوص الثابتة إلى قاموس الترجمة الشامل.
